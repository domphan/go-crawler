crawler instructions


use the form

url

input a valid url consisting of at least a scheme and authority

keyword

optional: enter a keyword to halt the search and highlight the node if discovered during crawl

crawl method

depth-first: starts at the base url then randomly selects urls to crawl until depth limit reached

breadth-first: starts at the base url then follows all links from the start page, and all links from each page visited until depth limit reached

cookies

url and keywords of past searches will be saved and suggested in the input boxes
to delete cookies, click the clear cookies button

manipulate the graph

zoom

mouse scrolling up and down will zoom in / out on the cursor inside the graph

pan

dragging empty space will move the screen to different parts of the graph

nodes

highlighting over nodes will display its url
clicking on a node will open a new tab to the url

drag & drop

drag on nodes to move them around to get a better view of the url

run our crawler locally

github repo

detailed instructions to set up your environment on the project repo
